# Python 기초 - 변수 선언 규칙과 기본 자료형

## 📚 개요

이 학습 자료는 Python 프로그래밍의 기초인 **변수 선언 규칙**과 **기본 자료형**(숫자형, 문자열)을 체계적으로 학습할 수 있도록 구성된 교육용 자료입니다.

### 🎯 학습 목표
- Python의 변수 선언 규칙과 네이밍 컨벤션 이해
- 숫자형 자료형(int, float, complex)과 연산 방법 습득
- 문자열 자료형과 다양한 문자열 처리 기법 학습
- 자료형 변환과 내장 함수 활용 능력 배양
- 효과적인 주석 작성 방법 이해

### 👥 대상
- Python을 처음 배우는 초보자
- 프로그래밍 경험이 없거나 적은 학습자
- Python 기초 문법을 체계적으로 정리하고 싶은 분

## 📌 주요 개념 요약

### 변수 (Variable)
```python
# 변수 선언
name = "홍길동"
age = 25

# 스네이크 케이스 사용
student_name = "김철수"
total_price = 15000
```

### 숫자형 자료형
```python
# 정수 (int)
count = 100

# 실수 (float)
pi = 3.14159

# 복소수 (complex)
complex_num = 3 + 4j
```

### 문자열 자료형
```python
# 문자열 선언
text = "Python"

# 인덱싱
first_char = text[0]  # 'P'

# 슬라이싱
substring = text[0:3]  # 'Pyt'

# f-string 포맷팅
name = "홍길동"
age = 25
message = f"제 이름은 {name}이고, {age}살입니다."
```

### 자료형 변환
```python
# 문자열 → 정수
num = int("123")

# 정수 → 문자열
text = str(456)

# 실수 → 정수
integer = int(3.14)
```

### 내장 함수
```python
# type() - 자료형 확인
type(100)  # <class 'int'>

# len() - 길이
len("Python")  # 6

# isinstance() - 자료형 검증
isinstance(100, int)  # True
```
