## 🎨 교육 방법론

### 1. 점진적 학습 (Scaffolding)
- 쉬운 개념부터 어려운 개념으로 단계적 진행
- 이전 섹션의 내용을 바탕으로 새로운 개념 소개
- 예제의 복잡도를 점진적으로 증가

### 2. 실습 중심 학습 (Learning by Doing)
- 모든 이론 섹션 뒤에 즉시 실습
- 예제 코드를 직접 타이핑하고 실행
- 연습 문제로 개념 확인

### 3. 반복 학습 (Repetition)
- 핵심 개념을 여러 예제에서 반복
- 이전 섹션의 개념을 다음 섹션에서 재활용
- 종합 실습에서 모든 개념 통합

### 4. 시각적 학습 (Visual Learning)
- 메모리 구조 다이어그램
- 인덱스 개념도
- 표를 통한 정리

### 5. 즉각적인 피드백 (Immediate Feedback)
- 예제 코드의 출력 결과 명시
- 연습 문제의 해설 제공
- 잘못된 예와 올바른 예 비교
